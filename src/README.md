# Cascara Theme for VS Code

![](https://sandydunlop.github.io/assets/images/vscode-theme-preview.png)


A dark theme designed for modern software development. Harmonized with the exact visual identity and palette of the upcoming **Cascara Studio** and its official platform website.

## Designed for Semantic Clarity

The Cascara Theme moves away from chaotic, high-contrast rainbow highlighting. Instead, it prioritizes a sophisticated, low-fatigue slate backdrop combined with an intentional, curated palette. Syntax colors are chosen strictly to map cleanly to the developer's mental model of the code's structure, allowing you to scan syntax hierarchies at a glance.

* **Deep, Premium Slate Canvas:** A carefully tuned dark background that minimizes eye strain during extended coding sessions or late-night release loops.
* **Vibrant, Controlled Accents:** Emerald greens, electric cyans, and warm ochres break through the dark canvas cleanly, calling immediate attention to keywords, string definitions, and variable operators.
* **Unified Ecosystem:** Maintain a seamless visual experience across your development tools. Transition flawlessly from the **Cascara Studio** desktop environment and website right into your favorite code editor.

---

## Installation

1. Open **Visual Studio Code**
2. Launch the Extension Marketplace (`Ctrl+Shift+X` or `Cmd+Shift+X`)
3. Search for `Cascara Theme`
4. Click **Install**
5. Select `Cascara OG` as your Color Theme
